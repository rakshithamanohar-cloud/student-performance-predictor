# Student Performance Predictor 🎓

A machine learning web app that predicts whether a student will be a high or low performer based on academic and socio-economic factors.

🚀 **Live App:** [Click here to try it]([https://your-app-url.streamlit.app](https://student-performance-predictor-oyr6pdbs7bfechdkm2c29w.streamlit.app/))

## About the Project

I built this as a hands-on ML project to understand how real-world data behaves and how different factors affect student outcomes. The dataset contains 1000 students from Bangladesh with features like study hours, attendance, family income, and previous GPA.

## What I Found

Family income turned out to be the biggest predictor of student success (19.42%), more than study hours or attendance. It made me realize that socio-economic background plays a bigger role in academic performance than effort alone.

## Models Used

- Logistic Regression
- Random Forest Classifier ✅ (best performer)

Final accuracy: **67%**
